# Module 1 · Lesson 04 Video Script — Formal Greetings by Time and Context

Updated: 2026-06-02 15:20 CEST

**Purpose:** Learners choose and answer safe greetings in formal A1 situations.

**Presenter language:** Boss records explanation later in Malayalam/Manglish. This script uses English stage directions plus exact German/on-screen text.

**Target length:** 35 minutes.

**Learner output by end:** Answer 8 greeting prompts using `Guten Morgen`, `Guten Tag`, `Guten Abend`, `Hallo`, and `Auf Wiedersehen` appropriately.

---

**Visual:** Language-school reception / Goethe Kochi practice desk. Clock shows 9:00.

**Audio / on-screen German:**

> Frau Weber: **Guten Morgen.**

**Practice pause:**

> Reply aloud: **Guten Morgen, Frau Weber.**

**Presenter:**

[Boss explains in Malayalam/Manglish: “Today we are not learning 25 greetings. We learn the safe set you can actually use.”]

## 1:00–5:30 — The safe greeting set

**Teaching point:** Formal greetings are safer with teachers, examiners, offices, doctors, counters, and unknown adults.

**On-screen table:**

| Context | German | Meaning |
|---|---|---|
| Morning | Guten Morgen | Good morning |
| Day / formal hello | Guten Tag | Good day/hello |
| Evening | Guten Abend | Good evening |
| Casual/neutral | Hallo | Hello |
| Leaving | Auf Wiedersehen | Goodbye |

**Presenter:**

[Boss explains in Malayalam/Manglish: If you are unsure in an exam or office, `Guten Tag` is usually a safe formal default.]

**Say-aloud drill:**

1. **Guten Morgen.**
2. **Guten Tag.**
3. **Guten Abend.**
4. **Hallo.**
5. **Auf Wiedersehen.**

## 5:30–9:30 — Morning: `Guten Morgen`

**Teaching point:** Use `Guten Morgen` in the morning. Reply with the same greeting.

**On-screen scene:**

```text
9:00 AM
Frau Weber: Guten Morgen.
You: Guten Morgen, Frau Weber.
```

**Presenter:**

[Boss explains in Malayalam/Manglish: Add `Frau Weber` when addressing the teacher politely. In Goethe speaking, polite clear greeting is useful.]

**Practice prompts:**

1. Teacher at 8 AM: **Guten Morgen.** → **Guten Morgen.**
2. Frau Weber at class start: **Guten Morgen.** → **Guten Morgen, Frau Weber.**
3. Examiner morning slot: **Guten Morgen.** → **Guten Morgen.**

---

## 9:30–14:00 — Day/formal: `Guten Tag`

**Teaching point:** `Guten Tag` is a strong safe default in formal day contexts.

**On-screen:**

```text
Goethe exam desk
Examiner: Guten Tag.
You: Guten Tag.
```

**Presenter:**

[Boss explains in Malayalam/Manglish: `Hallo` is not wrong everywhere, but for examiner/office/unknown adult, `Guten Tag` feels safer and more respectful.]

**Context sorting:**

Use `Guten Tag` for:

- Goethe examiner
- visa office counter
- doctor reception
- language-school office

Use `Hallo` for:

- friend
- classmate
- casual chat

**Practice pause:**

> Say a safe formal reply: **Guten Tag.**

---

## 14:00–18:30 — Evening: `Guten Abend` and the `Gute Nacht` trap

**Teaching point:** `Guten Abend` = greeting in the evening. `Gute Nacht` = when someone is going to sleep/leaving for the night, not a normal hello.

**On-screen split:**

```text
Meeting at 7 PM: Guten Abend. ✅
Going to sleep: Gute Nacht. ✅
Meeting at 7 PM: Gute Nacht. ❌
```

**Presenter:**

[Boss explains in Malayalam/Manglish: Malayali learners sometimes translate “good night” as evening greeting. In German, be careful.]

**Repair drill:**

Wrong:

> **Gute Nacht** at a 7 PM class.

Better:

> **Guten Abend.**

Repeat 3 times:

> **Guten Abend.**

---

## 18:30–22:30 — Leaving: `Auf Wiedersehen`

**Teaching point:** `Auf Wiedersehen` is formal/safe goodbye. `Tschüss` is casual.

**On-screen table:**

| Leaving whom? | Safer goodbye |
|---|---|
| Teacher/examiner/office | Auf Wiedersehen |
| Friend/classmate | Tschüss |

**Presenter:**

[Boss explains in Malayalam/Manglish: For A1 exam speaking, use the safer formal goodbye unless the scene is clearly casual.]

**Dialogue:**

```text
Frau Weber: Auf Wiedersehen.
You: Auf Wiedersehen, Frau Weber.
```

**Practice pause:**

> Say: **Auf Wiedersehen, Frau Weber.**

---

**Presenter:**

[Boss explains in Malayalam/Manglish: Notice the pattern: formal unknown adult = safer formal greeting. Friend/classmate = casual allowed.]

## 27:30–31:30 — Common mistakes and repair

**Mistake 1:** using `Gute Nacht` as evening hello.

Repair:

> **Guten Abend.**

**Mistake 2:** using `Hey` with examiner.

Repair:

> **Guten Tag.**

**Mistake 3:** forgetting goodbye.

Repair:

> **Auf Wiedersehen.**

**Practice pause:**

Say the formal safety trio:

```text
Guten Morgen.
Guten Tag.
Auf Wiedersehen.
```
