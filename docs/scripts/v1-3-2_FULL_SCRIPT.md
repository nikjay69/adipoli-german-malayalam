## Section 2: Formal Goodbyes (1:00–2:30)

### Visual: [HIGHLIGHT: Auf Wiedersehen with syllable breakdown, plus Auf Wiederhören]
### Narration:
"The classic, official German goodbye:

**Auf Wiedersehen!**
- Auf = until
- Wieder = again
- Sehen = seeing
- 'Until we see each other again!' Beautiful, right?

Pronunciation: 'Owf VEE-der-zay-en'

Use this with: strangers, elders, officials, formal situations. Like saying 'ningal' situations in Malayalam — whenever you'd be respectful.

Now here's a fun one:

**Auf Wiederhören!**
- Hören = hearing
- 'Until we hear each other again!'

When do you use this? **On the PHONE!** Because on the phone, you can't SEE the person — you can only HEAR them. So Germans switch 'sehen' to 'hören.' Clever, right?

⏸️ **[PAUSE — Student says: 'Auf Wiedersehen!' then 'Auf Wiederhören!']**

Rule: Face-to-face = Wiedersehen. Phone = Wiederhören. Simple.

*Phone-il 'Auf Wiederhören' — face-to-face 'Auf Wiedersehen.' Logic aanu!*"

## Section 3: Informal Goodbyes (2:30–4:00)

### Visual: [TABLE: Informal goodbye options with usage context]
### Narration:
"Now the casual ones — what you'll use 90% of the time with friends:

| German | Meaning | When to use |
|--------|---------|-------------|
| **Tschüss!** | Bye! | Universal casual — safe everywhere |
| **Tschau!** | Ciao! | Borrowed from Italian — very casual |
| **Bis bald!** | See you soon! | When you'll see them again sometime |
| **Bis morgen!** | See you tomorrow! | When you'll see them tomorrow |
| **Bis dann!** | See you then! | When you've set a time to meet |
| **Bis später!** | See you later! | When you'll see them later today |

The 'Bis' family is super useful:
- **Bis** = until/by
- **Bis + [when]** = see you [when]!

You can combine 'Bis' with almost anything:
- Bis Montag! (See you Monday!)
- Bis nächste Woche! (See you next week!)
- Bis zum Unterricht! (See you at class!)

⏸️ **[PAUSE — Student says: 'Tschüss! Bis morgen! Bis bald!']**

The most common casual goodbye? **Tschüss.** When in doubt, say Tschüss. It works with everyone except maybe the queen."

## Section 4: Night-Specific + Regional (4:00–5:00)

### Visual: [SLIDE: Moon icon with 'Gute Nacht!' plus regional Germany map]
### Narration:
"Two special cases:

**Gute Nacht!** — Good night!

Use this ONLY at actual bedtime or when someone is going to sleep. NOT as a general evening goodbye! If it's 8 PM and someone is leaving a restaurant, you say 'Tschüss' or 'Auf Wiedersehen' — NOT 'Gute Nacht.'

Bonus: **'Schlaf gut!'** = Sleep well! Sweet and friendly.

**Regional bonus** (just recognize these, don't worry about using them):
- **Servus!** — Bavaria/Austria (means both hello AND goodbye!)
- **Tschö!** — Rhineland version of Tschüss
- **Mach's gut!** — 'Take care!' (literally: do it well)

⏸️ **[PAUSE — Student says: 'Gute Nacht! Schlaf gut!']**

*'Gute Nacht' urakkathinu maatraam — 8 PM-nu restaurant-il ninnu pokunna aalodu 'Tschüss' parayan, 'Gute Nacht' alla!*"

## Section 5: The Danke-Bitte Dance (5:00–6:30)

### Visual: [TABLE: Thank you / You're welcome pairs with formality levels]
### Narration:
"Now the magic words — Danke and Bitte. Germans use these CONSTANTLY.

**Saying thank you:**
| German | Level | Use when... |
|--------|-------|-------------|
| **Danke!** | Standard | Everyday thanks |
| **Danke schön!** | Warm | Someone did something nice |
| **Vielen Dank!** | Formal/strong | You really mean it |
| **Danke sehr!** | Formal | Professional contexts |

**Responding to thanks:**
| German | Level | Meaning |
|--------|-------|---------|
| **Bitte!** | Standard | You're welcome |
| **Bitte schön!** | Warm | You're very welcome |
| **Bitte sehr!** | Formal | You're very welcome |
| **Gern geschehen!** | Polite | It was my pleasure |
| **Kein Problem!** | Casual | No problem! |

The dance: Someone says Danke → You say Bitte. Always.

But wait — **'Bitte' has THREE meanings!**
1. **You're welcome** — response to Danke
2. **Please** — 'Ein Kaffee, bitte!' (A coffee, please!)
3. **Here you go** — when handing something to someone

Context tells you which meaning. Don't overthink it.

⏸️ **[PAUSE — Student practices: 'Danke schön!' → 'Bitte schön!' / 'Vielen Dank!' → 'Gern geschehen!']**

*'Bitte' = please, you're welcome, AND here you go. Moonnu meaning oru word-il — context-il ninnu manasilaakunnu!*"

## Section 6: Sorry & Excuse Me (6:30–7:30)

### Visual: [HIGHLIGHT: Entschuldigung vs Es tut mir leid with situation icons]
### Narration:
"Two 'sorry' words — and they're different:

**Entschuldigung!**
Use for: getting attention, excusing yourself, small interruptions
- Bumping into someone on the street → 'Entschuldigung!'
- Getting a waiter's attention → 'Entschuldigung!'
- Asking a stranger for directions → 'Entschuldigung, wo ist der Bahnhof?'

Think of it as 'excuse me.'

**Es tut mir leid.**
Use for: genuine apology, expressing sorrow
- You broke something → 'Es tut mir leid!'
- You can't make it → 'Es tut mir leid, ich kann nicht kommen.'
- Hearing bad news → 'Oh, das tut mir leid.'

Think of it as 'I'm sorry.'

Quick version of Entschuldigung? **Tschuldigung!** — Germans shorten it in casual speech.

⏸️ **[PAUSE — Student says: 'Entschuldigung!' (to get attention) then 'Es tut mir leid.' (to apologize)]**

*'Entschuldigung' = 'excuse me' — attention edukaan. 'Es tut mir leid' = 'I'm sorry' — genuine apology.*"
