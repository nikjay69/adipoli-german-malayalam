## Section 2: The One Rule — Wo? vs. Wohin? (0:35–2:00)

🎬 [Energy: Clear, revelatory — this is THE unlock moment]

**NARRATOR:**
"Here's the entire rule in two words:

**Wo?** — Where? → Location → **Dativ**
**Wohin?** — Where TO? → Movement/Direction → **Akkusativ**

That's it. Every time you use one of these prepositions, ask yourself: Am I describing WHERE something IS? Or WHERE something is GOING?

Standing still? Dativ.
Moving to a new position? Akkusativ.

'Wo ist das Buch?' — 'Auf **dem** Tisch.' Location. Dativ.
'Wohin legst du das Buch?' — 'Auf **den** Tisch.' Direction. Akkusativ."

[VISUAL: Two-column diagram. Left: 🧍 person standing still = "Wo? → DATIV". Right: 🏃 person moving with arrow = "Wohin? → AKKUSATIV"]

**NARRATOR:**
"Malayalam-il ee concept implicit aanu — 'mēšayil irikkunnu' vs 'mēšayil vekkunnu.' Both use the same '-il' ending. But German SHOWS the difference by changing the article. Explicit aanu — you can SEE the meaning!"

⏸️ [PAUSE — 4 seconds] "Repeat after me: Wo? → Dativ. Wohin? → Akkusativ. Say it again — Wo? → Dativ. Wohin? → Akkusativ. Good."

## Section 3: Meet the Nine (2:00–3:15)

🎬 [Energy: Rhythmic, memorable — chant-like delivery]

**NARRATOR:**
"Nine prepositions switch cases. Only nine. Here they are:

**in** — in, into
**an** — at, on (vertical surface)
**auf** — on, onto (horizontal surface)

That's group one — the big three. You'll use these the most.

**über** — over, above
**unter** — under, below
**vor** — in front of
**hinter** — behind

Group two — spatial relationships.

**neben** — next to
**zwischen** — between

Group three — relative position.

Memory trick: **'In an auf — über unter — vor hinter — neben zwischen.'** Say it like a rhythm. In-an-auf, über-unter, vor-hinter, neben-zwischen."

## Section 4: IN — The Most Important One (3:15–4:45)

🎬 [Energy: Focused, step-by-step — showing both cases clearly]

**NARRATOR:**
"Let's see each case in action. Starting with **in** — the one you'll use most.

**Location (Wo? → Dativ):**
'Ich bin **im** Zimmer.' — I'm in the room.
'im' = in dem. Contracted. You'll ALWAYS say 'im,' never 'in dem.'

'Sie ist **in der** Küche.' — She's in the kitchen.
Küche is feminine → in der.

**Direction (Wohin? → Akkusativ):**
'Ich gehe **ins** Zimmer.' — I'm going into the room.
'ins' = in das. Contracted.

'Er geht **in die** Küche.' — He's going into the kitchen.
die stays die in Akkusativ for feminine."

[VISUAL: Room diagram. Person standing inside room = "im Zimmer (Dativ)". Person walking toward room with arrow = "ins Zimmer (Akkusativ)"]

⏸️ [PAUSE — 4 seconds] "Your turn: 'The cat is IN the kitchen.' Location — so Dativ. 'Die Katze ist...' ... '**in der** Küche!' in der — because Küche is feminine."

⏸️ [PAUSE — 4 seconds] "Now: 'The cat runs INTO the kitchen.' Direction — Akkusativ. 'Die Katze läuft...' ... '**in die** Küche!' in die."

**NARRATOR:**
"Same preposition. Same noun. Different case. The ONLY difference is: standing still or moving?"

## Section 5: AUF and AN (4:45–6:15)

🎬 [Energy: Building confidence — pattern recognition kicking in]

**NARRATOR:**
"**AUF** — on a horizontal surface. Think: tables, floors, shelves.

**Location:** 'Das Buch liegt **auf dem** Tisch.' — The book is lying on the table.
**Direction:** 'Ich lege das Buch **auf den** Tisch.' — I'm putting the book on the table.

See the pattern? dem → den. Dativ → Akkusativ. Same as **in**."

[VISUAL: Book on table (still) vs. hand moving book to table. Articles highlighted]

**NARRATOR:**
"**AN** — on a vertical surface or at an edge. Think: walls, doors, the sea.

**Location:** 'Das Bild hängt **an der** Wand.' — The picture hangs on the wall.
Wand is feminine → an der (Dativ).

**Direction:** 'Ich hänge das Bild **an die** Wand.' — I'm hanging the picture on the wall.
an die (Akkusativ).

Contractions for **an**: an dem = **am**. an das = **ans**.
'Ich stehe **am** Fenster.' — I'm standing at the window. Location, Dativ."

⏸️ [PAUSE — 4 seconds] "Quick drill: 'Das Bild hängt an ___ Wand.' Location or direction? ... Location! So Dativ. '**an der** Wand.' ✅"

⏸️ [PAUSE — 4 seconds] "Now: 'Ich hänge das Bild an ___ Wand.' ... Direction! Akkusativ. '**an die** Wand.' ✅"

## Section 6: The Other Six — über, unter, vor, hinter, neben, zwischen (7:15–8:30)

🎬 [Energy: Efficient, pattern-confirming — "same rule, more words"]

**NARRATOR:**
"The remaining six work EXACTLY the same way. I'll go faster because the rule hasn't changed.

**über** — over/above:
'Die Lampe hängt **über dem** Tisch.' (Location — Dativ)
'Ich hänge die Lampe **über den** Tisch.' (Direction — Akkusativ)

**unter** — under:
'Die Katze schläft **unter dem** Bett.' (Location)
'Die Katze läuft **unter das** Bett.' (Direction)

**vor** — in front of:
'Das Auto steht **vor dem** Haus.' (Location)
'Er fährt das Auto **vor das** Haus.' (Direction)"

[VISUAL: Quick animations for each: static scene (Dativ) → motion scene (Akkusativ)]

**NARRATOR:**
"**hinter** — behind:
'Der Garten ist **hinter dem** Haus.' (Location)

**neben** — next to:
'Ich sitze **neben der** Frau.' (Location — feminine Dativ = der!)

**zwischen** — between:
'Das Kind steht **zwischen den** Eltern.' (Location — plural Dativ = den!)"

[EXAM-TIP] "In Lesen, apartment ads use these constantly: 'Die Wohnung liegt neben dem Bahnhof' or 'zwischen der Schule und dem Park.' Knowing Dativ vs Akkusativ helps you READ faster — you instantly know if something is static or moving."

## Section 7: Contractions — The Spoken Shortcuts (8:30–9:00)

🎬 [Energy: Practical, "real German" emphasis]

**NARRATOR:**
"In spoken German, you MUST contract these. Nobody says 'in dem' — they say 'im.'

**in dem → im** — 'Ich bin im Haus.'
**in das → ins** — 'Ich gehe ins Haus.'
**an dem → am** — 'Ich stehe am Fenster.'
**an das → ans** — 'Geh ans Fenster!'

These four are mandatory. You'll hear them everywhere.

'Zum' and 'zur' from Module 16-1? Same idea — contractions make German flow naturally."

[VISUAL: Contraction table: in dem→im, in das→ins, an dem→am, an das→ans]
