**Topic: Past Passive (Präteritum Passiv)**
"Past Passive" (Ithu cheyyappettu / Athu cheyyappettirunnu).
Structure: [Wurde] + [Partizip II] (Past participle).
"Wurde" (Was / Were).

- "Das Haus wurde gebaut." (The house was built / Veedu paniyappettu).
- "Der Brief wurde geschrieben." (The letter was written).
- "Die Autos wurden repariert." (The cars were repaired).

"Wurde" ennathe word orma vekkuka. Past status!

**Topic: Future Passive (Futur Passiv)**
"Future Passive" (Ithu cheyyappedum).
Structure: [Wird] + [Partizip II] + [werden] (at the end).
"Wird ... werden" (Will be ...).

- "Das Haus wird gebaut werden." (The house will be built / Veedu paniyappedum).
- "Die Prüfung wird geschrieben werden." (The exam will be written).
- "Der Fehler wird korrigiert werden." (The mistake will be corrected).

Note: Futur-ul modal verb "werden" last position-il redundant aayittu oru thavana koodi undaavu.

**Topic: Passive with Modal Verbs (Modalverben im Passiv)**
If you use "must" or "can":
Structure: [Modal Verb] + [Partizip II] + [werden].

- "Das Fenster muss geputzt werden." (The window must be cleaned).
- "Der Brief soll geschrieben werden." (The letter should be written).
- "Das Auto kann repariert werden." (The car can be repaired).

Kerala-yil "Paniyappedanam" or "Cheyyappenedanam" common alla, but instead passive construction logic set aanu!

**Topic: Interaction Drills**
Machane, ippo njane oru model passives introduce cheyyaam.
"Der Apfel wird gegessen." (Present)
"Der Apfel wurde gegessen." (Past)
"Der Apfel wird gegessen werden." (Future)
"Der Apfel muss gegessen werden." (Modal)

Adholoke! Simple natural passive flow.
