# A1 Curriculum Audit — Adipoli German

_Last updated: 2026-03-22_

## Goal
Build a course that is:
1. engaging enough that Malayali learners keep going,
2. practical enough for life in Germany,
3. systematic enough to help learners **ace Goethe A1 / Start Deutsch 1**.

This audit focuses on **curriculum sequencing**, **exam coverage**, and **engagement quality**.

---

## Scientific lens used
Each module was evaluated against five questions:

1. **A1 relevance** — does this materially help an A1 learner?
2. **Sequencing** — is it placed at the correct stage?
3. **Exam usefulness** — does it help Hören / Lesen / Schreiben / Sprechen?
4. **Production practice** — does it train output, not just recognition?
5. **Engagement** — is it memorable, motivating, and Malayali-relevant without becoming cringe?

---

## Core finding
The repo has strong breadth, but the course path needed one important correction:

### Module 16 should not block exam prep
- **Module 16** contains A1+/A2 bridge topics (dative deep dive, reflexive verbs, subordinate clauses).
- Those are useful, but they are **not core prerequisites for Goethe A1 success**.
- Learners should be able to go from:
  - Modules 1–15 (A1 foundations)
  - directly into Modules 17–18 (Goethe exam prep)

### Action taken
- Reframed **Module 16** as **optional bridge content**.
- Adjusted curriculum logic so learners can progress from Module 15 to Module 17 without being forced through Module 16 first.
- Updated module copy to reflect this.

## Module-by-module evaluation

### Modules 1–3: Foundation
**Role:** onboarding, sounds, greetings, identity, numbers, time

**Strengths**
- strong learner motivation early
- good Malayali context and emotional hook
- useful survival basics for speaking and listening

**Risks**
- can become a little too expansive before enough retrieval practice
- some lessons may over-explain rather than force short productive output

**Exam value**
- very high for Sprechen Teil 1, form filling, basic listening, personal details

**Priority**
- keep as core A1 foundation

---

### Modules 4–7: Everyday communication
**Role:** family, description, verbs, routine, food, shopping, money

**Strengths**
- highly practical and exam-relevant
- strong usefulness for daily life in Germany
- supports core sentence production

**Risks**
- article/case burden must be carefully staged
- some vocabulary sets may need pruning if they feel decorative instead of high-frequency

**Exam value**
- very high for Lesen, Sprechen, and short message writing

**Priority**
- keep central; this is the engine room of A1 usability

---

### Modules 8–12: Living, travel, health, work, free time
**Role:** location, movement, travel, doctor, office/university, hobbies, weather

**Strengths**
- very realistic migration/life-in-Germany content
- broad enough to support exam topics and real interactions
- strong for vocabulary expansion without losing daily relevance

**Risks**
- easy to become too broad if every subtopic gets equal weight
- should continue favoring high-frequency structures over encyclopedic coverage

**Exam value**
- high across all four skills, especially reading/listening topics and short speaking prompts

**Priority**
- core A1 track

---

### Modules 13–15: Past, formal life, integration
**Role:** Perfekt intro, forms/offices, culture and integration

**Strengths**
- formal-life module is highly valuable for Indian migrants
- office/forms content has strong exam relevance for Schreiben Teil 1
- integration/culture content keeps motivation high and increases practical readiness

**Risks**
- Perfekt is useful, but should not overshadow present-tense mastery for A1 exam performance
- culture content should stay practical and not drift into trivia

**Exam value**
- medium to high; especially useful for exam confidence and real-world readiness

**Priority**
- core A1 path, but keep Perfekt at "functional A1" depth, not too advanced

---

### Module 16: A1+ / A2 bridge
**Role:** optional grammar strengthening

**Strengths**
- strong long-term value for learners continuing past A1
- can help ambitious learners build momentum toward A2

**Risks**
- if placed as mandatory before exam prep, it can overload learners
- dative/subordinate-clause emphasis is beyond what many A1 learners need immediately

**Exam value**
- supportive, but not core

**Priority**
- optional enrichment, not a gatekeeper

---

### Modules 17–18: Goethe A1 exam prep
**Role:** exam-specific training

**Strengths**
- dedicated exam preparation layer exists
- good sign that exam success is treated explicitly rather than assumed

**Risks**
- these modules must stay tightly aligned with actual exam behavior, not generic revision
- they should emphasize time management, answer strategy, and repeated task formats

**Exam value**
- critical

**Priority**
- should follow directly after the core A1 track

## High-level recommendations

### 1. Keep the A1 path clean
Recommended main path:
- Modules **1–15** = core A1 build
- Modules **17–18** = exam prep
- Module **16** = optional bridge after or alongside exam prep

### 2. Prefer exam-useful vocabulary over decorative vocabulary
If a word is low-frequency and not useful for:
- self-introduction
- daily life in Germany
- common exam themes
- practical listening/reading
then it should not dominate lesson time.

### 3. Force more output in every module
Each lesson should ideally train:
- one recognition task
- one guided production task
- one exam-style micro-output task

### 4. Keep Malayalam bridges explanatory, not distracting
Malayalam context is a major differentiator.
But the best use is:
- clarify grammar
- build memory hooks
- reduce fear
not endless jokes or detours.

### 5. Protect learner confidence
Goethe A1 rewards:
- short correct sentences
- clear personal information
- predictable structures
- task familiarity

The course should repeatedly reinforce that **simple and correct beats fancy and broken**.

## Bottom line
The course already has strong ambition and useful coverage.
The most important scientific correction was to make sure learners are not forced into **A1+/A2 bridge grammar before core Goethe A1 exam prep**.

That correction is now reflected in the curriculum logic and module framing.
