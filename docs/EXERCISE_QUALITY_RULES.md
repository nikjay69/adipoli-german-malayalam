# Exercise Quality Rules — Adipoli German

> Every exercise must make the student THINK in German, not just recognize patterns.

---

## Exercises That Add No Value (NEVER create these)

1. **"Match word to phonetic text"** — You cannot learn pronunciation by reading "Shurn". Phonetic approximations in English are always wrong for sounds like ö, ü, ch. If the goal is pronunciation, use audio or teach the mouth mechanics. Never oversimplify German sounds to English equivalents — ö is NOT "er", it's its own sound.

2. **"Match German word to English translation" (bare lists)** — This is a flashcard disguised as an exercise. It tests recognition memory, not understanding. It's not immersive. If you must test vocabulary, put it in a situation.

3. **Any exercise that could exist in any language app** — If you could swap "German" for "Spanish" and the exercise still works identically, it has no value specific to our course.

## What Makes an Exercise Valuable

- **It forces a decision that requires understanding** — not just pattern matching
- **It's grounded in a real situation** — "You're at the bakery" not "Match column A to column B"
- **It teaches a RULE or PATTERN** — not an isolated fact
- **The wrong answers are plausible and instructive** — the distractors teach something too
- **It builds toward production** — the student gets closer to actually USING German

## Good Exercise Patterns

| Instead of... | Do this... |
|---|---|
| Match "Wasser" → "water" | "Your flatmate left a note: 'Kein ___ im Kühlschrank.' What's missing?" |
| Match word to pronunciation text | "Someone says 'Tsait' — which word is it?" (recognition from context) |
| "Translate: Guten Morgen" | "It's 2 PM. Your colleague says 'Guten Morgen!' — what's weird about that?" |
| "Select the correct article: ___ Tisch" | "You're filling a form. It says 'Familienstand'. What do they want to know?" |

## The Test

Before adding any exercise, ask:
> "Would a student groan and skip this? Or would they actually think?"

If they'd groan, delete it.
