## Overview

Wave 5 modules in script priority order:
1. **Module 15** — German Culture & Integration (cultural fluency, social norms, slang — helps real-life integration)
2. **Module 16** — A1+ Bonus Bridge (Dativ deep dive, Wechselpräpositionen, reflexive verbs, subordinate clauses — A2 preparation)

**Wave 5 unifying theme:** These modules are *optional enrichment*. Module 15 helps students survive culturally in Germany; Module 16 prepares ambitious learners for A2. Scripts should feel like "bonus content" — rewarding for students who completed the core. Tone can be more relaxed, exploratory, and fun.

**Priority note:** Wave 5 is the LAST priority for script creation. Only produce these after Waves 1–4 scripts are trusted and stable. However, the architecture is documented here so it's ready when needed.

## Module 15: German Culture & Integration (Kultur)

### Teaching Strategy
Module 15 is a breather from grammar — pure cultural content. The approach:
- Documentary/conversational tone, not classroom-lecture
- Heavy Kerala ↔ Germany comparisons (the emotional core of this module)
- Practical survival information (Anmeldung, Versicherung, Mülltrennung)
- Slang lesson is a fun reward for students who made it this far
- Low production exercise density — this is about UNDERSTANDING, not PRODUCING

## Module 16: A1+ Bonus Bridge (Optional)

### Teaching Strategy
Module 16 is for ambitious students who want to start bridging to A2. It goes deeper into grammar that A1 only touched: full Dativ system, Wechselpräpositionen, reflexive verbs, and subordinate clauses (weil/dass/wenn/als/ob). The approach:
- Build on what students already know (Dativ from Module 9 "mit+Dativ", subordinate clauses from Module 12 "weil")
- Clear visual systems (colour-coded tables, sentence diagrams)
- Malayalam parallels where applicable (verb-at-end word order!)
- Explicitly frame as OPTIONAL: "This is A2 preview. You can pass A1 without this. But if you want to get AHEAD..."
- Reading comprehension lesson as culmination — applying everything to real texts
