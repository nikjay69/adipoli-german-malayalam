## Golden Self-Introduction Template

This template should be the centerpiece of Module 2 and referenced throughout the course:

```
Guten Tag! Ich heiße [Name]. / Mein Name ist [Name].
Ich bin [age] Jahre alt.
Ich komme aus [city/country]. Ich bin [nationality].
Ich wohne in [city].
Ich bin [profession] von Beruf. / Ich studiere [subject].
Ich spreche [languages].
Mein Hobby ist [hobby]. / Ich [verb] gern.
```

**This 7-line template should be:**
- Introduced in Module 2
- Practiced in /practice/intro
- Referenced back in Module 11 (profession detail), Module 12 (hobbies detail)
- Tested in Module 18 (Sprechen Teil 1 mock)

## Common Malayali Learner Errors

1. **Confusing country/nationality/language:** "Ich komme aus Indien" ✅ but "Ich bin Indian" ❌ (should be "Inder/Inderin")
2. **Saying profession in English:** "Ich bin nurse" instead of "Ich bin Krankenschwester/Krankenpfleger"
3. **Weak nationality forms:** missing masculine/feminine distinction
4. **Skipping age:** leaving out "Jahre alt" → "Ich bin fünfundzwanzig" (acceptable but "Jahre alt" is safer)
5. **Name confusion:** Vorname vs Nachname — Kerala naming conventions don't map 1:1
6. **Freezing on "Buchstabieren":** haven't practiced spelling their own name
