# Module 02: Who Are You? (Wer bist du?)

## Teaching Goal
After this module, students should be able to **introduce themselves fully** (name, origin, profession, languages), **ask others for personal information**, **spell their name**, and **sustain a short self-introduction conversation**. This is the single most exam-critical personal module — it maps directly to Sprechen Teil 1 and Schreiben Teil 1.

## A1 Exam Relevance

| Exam Section | Relevance |
|---|---|
| **Sprechen Teil 1** | DIRECT — "Sich vorstellen" is exactly this module's output goal. Students present name, origin, job, languages, hobbies. |
| **Schreiben Teil 1** | DIRECT — form fields (Vorname, Nachname, Beruf, Staatsangehörigkeit, Sprachen) come from this vocabulary. |
| **Hören** | Short personal-info dialogues are common in Hören Teil 1 and 2. |
| **Lesen** | Self-descriptions, registration forms, and personal profiles appear in reading tasks. |

## Essential Outcomes
Students must be able to **produce** (not just recognize):
1. A complete self-introduction in 5-7 sentences: name, origin, residence, profession, languages
2. Ask someone their name, origin, and profession in both formal and informal register
3. Spell their own name aloud using the German alphabet
4. State their nationality correctly (Ich bin Inder/Inderin)
5. Conjugate "sein" for ich/du/er/sie and "sprechen" for common persons
6. Handle a 1-minute back-and-forth identity conversation without freezing

## Gap Analysis
- **No speaking exercises** (0/40) — catastrophic for the module most tied to Sprechen Teil 1
- **No form-filling practice** — this module should simulate Schreiben Teil 1 form fields
- **No free-text self-introduction** writing task
- **No 2-minute speaking drill** — needs timed self-introduction practice
- **No dictation** — listening to personal details and writing them down
- **Alphabet drilling is video-only** — needs interactive letter-by-letter spelling exercises
- **Profession gender forms** need more systematic practice (der Ingenieur / die Ingenieurin pattern)
- **Missing bridge to Module 14** (forms) and Module 18 (exam Sprechen Teil 1)

## Future Script Direction

### Suggested Narrative Approach
- Frame this as "the most important 5 sentences you'll ever learn in German"
- Build a "golden self-introduction" that students memorize first, then personalize
- Use progressive layering: name only → name + origin → + profession → + languages → full intro
- End with a real recording challenge

### Malayalam Bridge Points
- **Kerala names in German** — "Germans will call you 'A-kil' not 'Akhil'" — normalize pronunciation differences
- **Multiple languages already spoken** — Malayalis already speak 2-3 languages; German is just adding one more
- **"Nee evidunna?" → "Woher kommst du?"** — exact same social function
- **Profession culture** — Malayali career awareness (engineering, nursing, IT) maps directly to German Beruf vocabulary
- **Family pride in education/work** — motivational hook for mastering profession vocabulary

### Key Example Sentences/Dialogues to Feature
```
Golden Self-Introduction:
"Hallo! Ich heiße Arun. Ich komme aus Indien, aus Kerala.
Ich wohne jetzt in München. Ich bin Ingenieur von Beruf.
Ich spreche Malayalam, Englisch und ein bisschen Deutsch.
Ich lerne Deutsch seit drei Monaten. Freut mich!"

Formal Version:
"Guten Tag. Mein Name ist Arun Kumar.
Ich komme aus Indien. Ich wohne in München.
Ich bin Ingenieur. Ich spreche Englisch und lerne Deutsch.
Freut mich, Sie kennenzulernen."

Question-Answer Pairs:
Q: Wie heißen Sie? → A: Ich heiße Lakshmi.
Q: Woher kommen Sie? → A: Ich komme aus Indien.
Q: Was sind Sie von Beruf? → A: Ich bin Krankenschwester.
Q: Welche Sprachen sprechen Sie? → A: Ich spreche Malayalam und Englisch.
```
