## The "Golden Restaurant Dialogue" — Module 6 Centerpiece

This dialogue pattern (taught in v6-3-1) should be the module's signature output:

```
Kellner: Guten Tag! Was möchten Sie trinken?
Gast: Einen Kaffee, bitte.
Kellner: Und zum Essen?
Gast: Ich nehme eine Suppe, bitte.
Kellner: Gern. Noch etwas?
Gast: Nein, danke. Was kostet das?
Kellner: Acht Euro fünfzig.
Gast: Hier, bitte. Danke schön!
```

Key patterns to drill:
- `Ich nehme + ACCUSATIVE` (einen/eine/ein)
- `Ich hätte gern + ACCUSATIVE`
- `Was kostet...?`
- polite `bitte` placement

## Common Malayali Learner Errors

1. **Dropping accusative change:** "Ich nehme ein Kaffee" instead of "einen Kaffee"
2. **"gern" placement:** "Ich gern esse" instead of "Ich esse gern"
3. **English food words:** "Ich möchte rice" instead of "Reis"
4. **Price confusion:** "drei fünfzig" — is it 3.50 or 350?
5. **Too polite (Indian style):** overly long phrases when "Einen Kaffee, bitte" suffices
6. **Forgetting articles:** German food always has an article; memorize der/die/das WITH the word
