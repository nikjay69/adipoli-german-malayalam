## The 20 Essential A1 Verbs (Module 5 Core)

These should be automatic by Module 5 completion. Script v5-1-2 drills all 20:

**Daily life:** machen, gehen, kommen, essen, trinken
**Communication:** sprechen★, lesen★, schreiben, hören, fragen
**Movement/routine:** fahren★, laufen★, schlafen★, arbeiten✦
**Useful:** brauchen, kaufen, wohnen, lernen, spielen, kochen

★ = stem-changing (du sprichst, du liest, du fährst, du läufst, du schläfst)
✦ = extra -e- needed (du arbeitest, du findest)

**Script must treat stem-changers and -eit- verbs as explicit warning zones.**

## Common Malayali Learner Errors

1. **Missing -st ending:** "du mache" instead of "du machst"
2. **Forgetting to split separable verbs:** "Ich aufstehe" instead of "Ich stehe auf"
3. **Stem-change ignorance:** "er spreche" instead of "er spricht"
4. **English word order with separable verbs:** carrying SVO into German
5. **Extra -e- forgotten:** "du arbeitt" instead of "du arbeitest"
6. **Time expression placement:** putting time at end instead of German TMP order
