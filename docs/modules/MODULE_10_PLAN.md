## Health Complaint Pattern System

This is the core grammar/vocabulary pattern for the module:

### Pattern A: "Ich habe + Symptom"
```
Ich habe Kopfschmerzen.       (headache)
Ich habe Bauchschmerzen.      (stomach ache)
Ich habe Rückenschmerzen.     (back pain)
Ich habe Halsschmerzen.       (sore throat)
Ich habe Zahnschmerzen.       (toothache)
Ich habe Fieber.              (fever)
Ich habe Husten.              (cough)
Ich habe Schnupfen.           (cold/runny nose)
```

### Pattern B: "Mir ist + adjective" (Dativ feeling)
```
Mir ist schlecht.              (I feel nauseous)
Mir ist schwindelig.           (I feel dizzy)
Mir ist kalt/warm.             (I'm cold/warm)
Mir ist übel.                  (I feel sick to my stomach)
```

### Pattern C: "Mein/Meine + body part + tut weh"
```
Mein Kopf tut weh.            (My head hurts)
Mein Bauch tut weh.           (My stomach hurts)
Meine Hand tut weh.           (My hand hurts)
Mein Knie tut weh.            (My knee hurts)
```

### Pattern D: Duration + frequency
```
Seit gestern / seit zwei Tagen / seit einer Woche
Ich habe seit drei Tagen Fieber.
Die Schmerzen sind seit gestern schlimmer.
```

**All four patterns should be drilled. Pattern A is the most A1-safe and should be automatic.**

## Golden Doctor Dialogue

This dialogue should be the centerpiece of Module 10:

```
Arzt:  Guten Tag! Was kann ich für Sie tun?
Patient: Guten Tag! Ich habe starke Kopfschmerzen und Fieber.
Arzt:  Seit wann haben Sie die Beschwerden?
Patient: Seit zwei Tagen.
Arzt:  Haben Sie auch Husten oder Schnupfen?
Patient: Ja, ich habe auch Husten.
Arzt:  Nehmen Sie Medikamente?
Patient: Nein, keine Medikamente.
Arzt:  Ich verschreibe Ihnen Schmerztabletten. Nehmen Sie dreimal täglich eine Tablette.
Patient: Vielen Dank, Herr Doktor.
Arzt:  Gute Besserung! Kommen Sie wieder, wenn es nicht besser wird.
```

### Golden Pharmacy Dialogue
```
Kunde: Guten Tag! Haben Sie etwas gegen Kopfschmerzen?
Apotheker: Ja, möchten Sie Ibuprofen oder Paracetamol?
Kunde: Ibuprofen, bitte. Wie oft soll ich die Tabletten nehmen?
Apotheker: Dreimal am Tag, nach dem Essen. Nicht mehr als drei Tabletten pro Tag.
Kunde: Was kostet das?
Apotheker: Fünf Euro neunzig.
Kunde: Danke schön!
Apotheker: Gute Besserung!
```

## Common Malayali Learner Errors to Script Against

1. **Using English medical terms:** "Ich habe fever" instead of "Ich habe Fieber" — many medical terms feel "international" but German has its own
2. **Article confusion with body parts:** Body parts have mixed genders (der Kopf, die Hand, das Auge) — no pattern from Malayalam helps
3. **"Mir ist" construction unfamiliar:** "Mir ist schlecht" — dative construction feels unnatural; students default to "Ich bin schlecht" (which means "I am bad/evil")
4. **Confusing -schmerzen compounds:** Kopfschmerzen, Bauchschmerzen, Halsschmerzen — the compound structure is powerful once understood
5. **"Seit" + present tense trap:** German says "Ich habe seit zwei Tagen Fieber" (present tense with seit); English/Malayalam would use past "I have had fever for two days"
6. **Missing "Gute Besserung":** Not using this standard phrase (like "Get well soon") — it's expected in German social interaction
7. **Pharmacy vs Drogerie confusion:** Apotheke = prescription pharmacy; Drogerie (dm, Rossmann) = drugstore for non-prescription items
8. **Emergency number confusion:** 112 in Germany, not 100 or 101 (India) — must be memorized

## Malayalam Bridge Points

- **Ayurveda/home remedies → German pharmacy:** "In Kerala, amma gives you kashayam. In Germany, you go to the Apotheke. Both work, but you need to know the German way too!"
- **Doctor respect culture:** "In Kerala, we also respect doctors — 'Doctor-e kaanan' is always a formal visit. Same in Germany: use 'Sie' and 'Herr Doktor / Frau Doktor'."
- **Compound word power:** "German loves compound words. Kopf (head) + Schmerzen (pain) = Kopfschmerzen. Bauch (belly) + Schmerzen = Bauchschmerzen. Once you know this trick, you can build new words!"
- **Health insurance reality:** "In Germany, health insurance is mandatory — Krankenversicherung. This is different from India but it means doctor visits are mostly free. Bring your Versicherungskarte!"
- **112 emergency drill:** "Write 112 on a sticky note in your wallet. Not 100 (police India), not 108 (ambulance India), not 101. In Germany, ONE number: 112 for everything."
