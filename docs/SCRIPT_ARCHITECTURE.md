## 1. Purpose of This Document

The Adipoli German course is not just a library of videos. Every script must do 4 jobs at once:

1. **Teach clearly** — explain German in a way a Malayalam-speaking beginner understands
2. **Prepare for Goethe A1** — map each lesson to a real Hören/Lesen/Schreiben/Sprechen need
3. **Create reusable assets** — scripts should work for video, audio, app text, and revision handouts
4. **Lead into practice** — every script should feed exercises, not end as passive content

### Core principle
**Script-first, not video-first.**

The script is the real teaching asset. Video is a delivery format. A strong script can become:
- a narrated lesson video
- an audio lesson
- an in-app reading lesson
- a revision PDF
- a speaking drill prompt
- a writing prompt generator

## 2. The Golden Script Philosophy

Every successful Adipoli German script should feel like this:
- **Warm and motivating** like a Malayali teacher or older cousin
- **Structured and exam-aware** like a serious A1 coach
- **Lightly funny** without becoming cringe
- **Practical** — always connected to real life in Germany
- **Safe for A1** — short, correct, high-probability patterns over flashy language

### What scripts should NOT do
- Over-explain grammar without usable examples
- Drift into A2+ unless clearly marked
- Use Malayalam bridges so heavily that German disappears
- Become motivational fluff with no production outcome
- Teach language that is “interesting” but not exam-safe or life-useful

```md

A1 Exam Link: [which exam section this supports]

[Course intro animation / branding]
"Willkommen bei Adipoli German! [Lesson topic teaser]"

### Visual: [describe what is on screen]

"[Full narration text in quotes. Malayalam/Manglish bridge lines in italics.]"
### Key Vocabulary Overlay:
[word — translation]

## Section 2: Core Teaching (1:30–6:00)
### Visual:
[table / diagram / dialogue / form / illustration]
### Narration:
"[Teaching content with clear examples and pauses marked as ⏸️]"

⏸️ [what the student should try]

### Visual:
[examples appearing on screen]

[next lesson teaser]
```

- Module number
- Lesson number

## 5. The Ideal Internal Structure of a Script

Every strong Adipoli German script should move through this teaching arc:

### 1. Hook
Answer: **Why should I care?**

Examples:
- “This is the easiest writing mark in the exam.”
- “If you can say this, you can survive your first German café visit.”
- “Malayalis usually get confused here — let’s fix that now.”

### 2. Pattern
Answer: **What is the German rule or sentence pattern?**

Examples:
- ich komme aus + place
- Ich hätte gern + Akkusativ
- haben/sein + Partizip II

### 3. Meaning
Answer: **What does it mean in real life?**

Examples:
- introducing yourself
- buying a ticket
- filling out a form
- talking about yesterday

### 4. Contrast
Answer: **Where will Malayali learners likely go wrong?**

Examples:
- halb drei ≠ 3:30
- Staatsangehörigkeit ≠ country name
- der/die/das does not map to Malayalam gender logic
- German word order in main clause vs subordinate clause

### 5. Drill
Answer: **Can the student produce it immediately?**

Examples:
- say it
- repeat it
- fill it
- transform it
- answer a question with it

### 6. Transfer
Answer: **Can the student use it in the exam or real life?**

Examples:
- write a short message
- answer a Sprechen card
- identify it in Hören
- fill a real form field

### 7. Wrap
Answer: **What must the student remember today?**

Use a “3 takeaways” summary whenever possible.

## 6. Malayalam Bridge System

Malayalam bridges are one of the course’s biggest advantages — but only if used well.

### The purpose of Malayalam bridges
Malayalam should help learners:
- notice what is familiar
- avoid common transfer mistakes
- stay emotionally connected
- remember patterns faster

Malayalam is **not** there to replace the German.

### The 5 Malayalam bridge types

#### A. Confidence bridge
Use when the learner already knows something similar.

Examples:
- “You already know the sch sound from Malayalam ശ/ഷ.”
- “Malayalam also changes verb forms. So conjugation is not a new concept.”

#### B. Trap bridge
Use when Malayalam/English interference causes mistakes.

Examples:
- “In Malayalam, you may not think about article gender. In German, you must.”
- “Don’t translate directly from English here.”

#### C. Structure bridge
Use when Malayalam word order or grammar can explain German clearly.

Examples:
- subordinate clause verb-final order in German compared with normal Malayalam clause flow
- dative idea compared loosely to Malayalam -ക്ക് / -നു style recipient meaning

#### D. Cultural bridge
Use when a Kerala situation helps make the German scene vivid.

Examples:
- filling a German Anmeldung form compared to filling a government form in Kerala
- restaurant ordering compared to asking in a hotel/thattu kada setting
- punctuality compared with “IST” jokes

#### E. Memory bridge
Use when a Malayalam phrase helps learners remember a German rule.

Examples:
- “halb drei trap” explained through repeated Malayalam warning phrasing
- “verb frame opens and closes” with a vivid local metaphor

### Malayalam bridge rules
1. **Keep them short** — one or two lines, not long detours
2. **Always return to German quickly**
3. **Use Malayalam where it adds clarity, not decoration**
4. **Prefer spoken-natural Manglish/Malayalam flavor over stiff textbook Malayalam**
5. **Never use Malayalam so heavily that non-Malayalam editors cannot understand the script**

### Good Malayalam bridge example
> “Familienname means your surname — *nammude veettu peru*. Vorname means your own given name. Don’t swap them.”

### Bad Malayalam bridge example
> A long unrelated Malayalam anecdote that delays the German point by 40 seconds.

## 7. Cultural Hook Patterns

A strong Adipoli German script often opens with a cultural hook. This helps memory, motivation, and emotional engagement.

### Hook Pattern 1: Kerala ↔ Germany contrast
Examples:
- IST vs German punctuality
- Kerala joint family vs German nuclear family
- hot fresh dinner vs Abendbrot
- KSRTC vs Deutsche Bahn

### Hook Pattern 2: Real immigrant survival moment
Examples:
- first visit to Ausländerbehörde
- ordering in a café
- introducing yourself in class
- reading a flat advertisement

### Hook Pattern 3: Exam payoff hook
Examples:
- “This is the easiest mark in Schreiben.”
- “This pattern appears in almost every Sprechen intro.”
- “If you master this audio trap, Hören gets much easier.”

### Hook Pattern 4: Common Malayali learner pain point
Examples:
- “Why does halb drei mean 2:30?”
- “Why does German split verbs?”
- “Why are there three words for ‘the’?”

### Hook Pattern 5: Identity and aspiration
Examples:
- “Thousands of Malayalis are already working in Germany.”
- “This one phrase can help you survive your first doctor visit.”
- “This is how you sound confident on day one.”

### Cultural hook guidelines
- Use one major hook per video
- Don’t stack too many jokes
- Hook must lead into actual teaching, not delay it
- If a hook is funny, it must still be respectful

## 8. Script Length Guidelines by Lesson Type

Not all lessons should feel the same. Different lesson types need different pacing.

### A. Motivation / onboarding videos
**Examples:** Why Learn German, Malayalis in Germany
- Target: **8–10 min**
- Hook-heavy, emotionally energizing
- Less drill-heavy
- Must still teach 3–5 anchor phrases or concepts
- Use stories, career pathways, migration reality checks

### B. Pronunciation videos
**Examples:** German Sounds, tricky sounds, alphabet
- Target: **10–12 min**
- High repetition
- Short explanation, many drills
- Must include listen-and-repeat blocks
- Avoid dense theory

### C. Grammar core videos
**Examples:** Present tense, Perfekt, possessives, separable verbs
- Target: **10–12 min**
- Pattern → table → examples → drills → production prompt
- One grammar idea per video whenever possible
- Minimum 3 practice pauses
- Use color-coded visual logic

### D. Vocabulary/theme videos
**Examples:** family, food, body parts, transport
- Target: **8–10 min**
- Organize vocab by scene or life context, not random lists
- Should always move from naming → using → mini-dialogue
- Include article practice wherever possible

### E. Dialogue / situation videos
**Examples:** restaurant, doctor, office, ticket counter
- Target: **9–11 min**
- Use a “golden dialogue” structure
- First: watch/listen
- Second: explain patterns
- Third: repeat and adapt
- End with substitution drill

### F. Exam strategy videos
**Examples:** Goethe format, writing part strategy, reading traps
- Target: **8–10 min**
- Very direct, low-fluff
- Show scoring and traps clearly
- Must include explicit exam advice markers
- Use model tasks and fast analysis

### G. Writing videos
**Examples:** form-filling, short messages, invitations
- Target: **9–10 min**
- Template-focused
- Show wrong vs right answers
- Include one full worked example
- End with a writing challenge linked to an exercise

### H. Reading comprehension videos
**Examples:** Wohnungsanzeigen, notices, ads
- Target: **8–10 min**
- Teach scanning, keywording, elimination
- Show real text formats
- Avoid over-translating every word
- Focus on how to find the answer fast

### I. Listening videos
**Examples:** Hören drills, announcements
- Target: **8–10 min**
- Audio first
- Transcript hidden initially
- Teach what to listen for, not just answers
- Include replay discipline similar to exam behavior

### J. Culture / enrichment videos
**Examples:** festivals, slang, customs
- Target: **8–10 min**
- More documentary/conversational tone
- Still teach usable phrases or survival knowledge
- Keep it practical, not just entertaining

## 9. How Scripts Should Connect to Exercises

This is one of the most important rules in the entire system:

> **Every script must point directly into exercise behavior.**

If a lesson video teaches something, the student should use that exact pattern in the next exercise set.

### Script-to-exercise connection model

#### Step 1: Teach a safe pattern
Example:
- “Ich heiße…”
- “Ich komme aus…”
- “Ich hätte gern…”
- “Ich bin um 7 Uhr aufgestanden.”

#### Step 2: Drill recognition
Examples:
- multiple-choice
- matching
- fill-blank
- ordering

#### Step 3: Drill controlled production
Examples:
- say the sentence aloud
- complete a dialogue
- write one missing word
- transform pronoun/verb/time

#### Step 4: Drill freer production
Examples:
- write a short self-introduction
- record your answer
- fill out a form
- write a 30-word message

### Exercise design rules for script writers
For every video, script writers should specify:
- **What 2–3 patterns are being trained**
- **What exact exercise types should follow**
- **What the learner should be able to produce by the end**

### Best mapping by lesson type

| Lesson type | Best exercise follow-up |
|-------------|-------------------------|
| Pronunciation | speaking + dictation |
| Grammar | fill-blank + sentence building + free-text |
| Vocabulary | matching + article drills + mini production |
| Dialogue | role-play + substitution drill + speaking |
| Writing | free-text + model answer comparison |
| Listening | dictation + comprehension questions |
| Reading | scanning questions + true/false + short written answers |
| Exam prep | timed mock tasks |

### Example: good script-exercise linkage
**Script teaches:** “Ich hätte gern einen Kaffee.”

**Exercises immediately after:**
1. Fill blank: Ich hätte ___ Kaffee.
2. Matching: food item ↔ correct article
3. Role-play: order 3 items
4. Free-text: write your café order
5. Speaking: say your order aloud

That is the ideal teaching loop.

## 10. Production Exercise Integration Rules

The course currently has recognition-heavy exercises. Scripts should be written with future production exercise integration in mind.

### The 3 production exercise layers

#### A. Free-text
Used for:
- self-introductions
- message writing
- family descriptions
- daily routine descriptions
- past tense writing

**Script implication:**
Every relevant video should end with at least one “Now write your own version” prompt.

#### B. Dictation
Used for:
- numbers
- dates
- prices
- short listening phrases
- Hören prep

**Script implication:**
Whenever a lesson teaches sound-sensitive material, include a dictation-ready phrase bank.

#### C. Speaking / recording
Used for:
- greetings
- self-introduction
- asking questions
- requests
- doctor/restaurant role-plays

**Script implication:**
Speaking lessons should include repeat-after-me and short independent production moments.

## 13. Reusable Scene Patterns

These scene types should repeat across the course to make production scalable.

### Pattern A: Golden dialogue
Use for:
- restaurant
- doctor
- office
- ticket counter
- greeting scenes

Structure:
1. full dialogue once
2. line-by-line breakdown
3. repeat-after-me
4. substitution drill
5. student role-play challenge

### Pattern B: Form walkthrough
Use for:
- Anmeldung
- exam forms
- doctor forms
- library cards

Structure:
1. explain the field
2. show common wrong answer
3. show correct answer
4. student fills one example
5. quick-fire additional forms

### Pattern C: Grammar pattern reveal
Use for:
- verb endings
- possessives
- Perfekt
- prepositions

Structure:
1. pattern headline
2. color-coded table
3. 3 examples
4. contrast with common mistake
5. controlled drill
6. mini production

### Pattern D: Reading decode
Use for:
- ads
- notices
- apartment listings
- emails

Structure:
1. show full text
2. identify target keywords
3. ignore noise
4. answer one sample question
5. timed second example

### Pattern E: Listening trap decode
Use for:
- announcements
- numbers
- dates
- prices
- short conversations

Structure:
1. listen once
2. what keyword mattered?
3. where was the trap?
4. replay
5. answer and explain

## 14. A1 Safety Rules for Script Content

To protect exam performance, scripts should train **safe output**.

### Favor:
- short correct sentences
- repeatable templates
- high-frequency vocabulary
- predictable sentence frames
- polite, standard register

### Avoid over-emphasizing:
- idioms too early
- slang outside cultural lessons
- fancy synonyms
- long compound sentences
- rare vocabulary that won’t help in exam or daily life

### Ideal A1-safe examples
- Ich heiße Nikhil.
- Ich komme aus Indien.
- Ich wohne in Kochi.
- Ich spreche Malayalam und Englisch.
- Ich hätte gern einen Kaffee.
- Ich bin um sieben Uhr aufgestanden.

## 15. Quality Checklist Before Marking a Script “Done”

### Pedagogy
- [ ] Teaching goal is explicit
- [ ] Script teaches one clear outcome, not too many competing outcomes
- [ ] Explanation is understandable for a true beginner
- [ ] At least 3 model examples are included
- [ ] At least 3 practice moments are included

### Malayalam bridge quality
- [ ] Malayalam support feels natural, not forced
- [ ] Malayalam lines increase clarity, not noise
- [ ] German stays the main learning language

### Exam readiness
- [ ] A1 exam section is clearly linked
- [ ] Pattern is A1-safe
- [ ] Any exam traps are named explicitly
- [ ] Output aligns with Goethe task expectations

### Production readiness
- [ ] Visual tags are present and usable
- [ ] Timing is realistic
- [ ] Script can be narrated smoothly
- [ ] Vocabulary overlays are specified
- [ ] Exercise tie-ins are named

### Language quality
- [ ] German examples are correct
- [ ] English explanation is clean
- [ ] No accidental A2+ overload
- [ ] Register is appropriate

## 16. Recommended Workflow for Script Creation

### Stage 1: Pull from source content
Read:
- module source file in `src/lib/content/modules/module-XX.ts`
- `MODULE_BLUEPRINTS.md`
- relevant wave architecture doc

### Stage 2: Define the teaching promise
Write in one sentence:
- “After this video, the student can…”

### Stage 3: Build the hook and scene flow
Use one cultural or exam hook.
Then map sections:
- hook
- teach
- drill
- transfer
- summary

### Stage 4: Add Malayalam bridge moments
Add 2–5 maximum.
Only where clarity or memory improves.

### Stage 5: Add exercise handoff
State exactly what the student should do next:
- exercise set
- speaking drill
- writing prompt
- mock task

### Stage 6: Validate for A1 safety
Trim anything too advanced.
Prefer simpler correct German.
