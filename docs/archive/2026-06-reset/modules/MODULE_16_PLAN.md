## Key Dialogues & Script Scenes to Feature

### Golden Dative Dialogue
```text
A: Kannst du mir helfen?
B: Ja, natürlich. Wobei denn?
A: Mein Deutschbuch gehört meiner Freundin, aber ich habe es.
B: Ah, dann gib es ihr morgen.
```
**Teaching point:** `mir`, `meiner Freundin`, `ihr` feel useful in real communication, not abstract.

### Wo vs Wohin Contrast Scene
```text
Das Buch liegt auf dem Tisch.
Ich lege das Buch auf den Tisch.

Wir sind im Zimmer.
Wir gehen ins Zimmer.
```
**Teaching point:** same preposition, different case because of location vs movement.

### Reflexive Routine Scene
```text
Ich wasche mich.
Dann ziehe ich mich an.
Ich setze mich an den Tisch.
Ich freue mich auf den Tag.
```
**Teaching point:** reflexive verbs are easiest to learn as a daily-routine chain, not isolated grammar facts.

### Controlled weil-Clause Scene
```text
Ich lerne Deutsch, weil ich in Deutschland arbeiten möchte.
Ich bleibe zu Hause, weil ich krank bin.
```
**Teaching point:** learners need 2-3 powerful sentence frames, not a giant clause lecture.

### Reading Strategy Script Frame
```text
Frage zuerst: Was suche ich?
Dann: Wo sind Zeit, Ort, Preis, Person?
Nicht jedes Wort übersetzen.
Zuerst Schlüsselwörter finden.
```
**Teaching point:** this module should train calm strategic reading, not perfectionism.

## Common Learner Errors to Script Against

1. **Using accusative after dative verbs** — `Ich helfe dich` instead of `Ich helfe dir`
2. **Confusing `der` as feminine nominative vs dative feminine** — chart must be very clear
3. **Memorizing prepositions without context** — learners need `wo` / `wohin` logic, not raw lists
4. **Forgetting movement vs location** — biggest trap in two-way prepositions
5. **Treating reflexive verbs as random** — teach in routine clusters for retention
6. **Panic around verb-final clauses** — reduce fear; do not flood with edge cases
7. **Reading every word** — this is exactly the habit Module 16 should break
8. **Believing Module 16 is required for A1** — docs must explicitly prevent this misunderstanding
