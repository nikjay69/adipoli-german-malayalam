# Script Quality Standard — Adipoli German

> Reference for all video script quality passes. Scripts meeting these standards earn ✅ DONE status.
> Created: 2026-03-23 (quality pass cron)

---

## Mandatory Checklist (every script must pass all)

### Structure & Pacing
- [ ] **Hook under 60 seconds** — states concrete payoff (exam %, real-life scenario, or "after this you can X")
- [ ] **No monologue block exceeds 90 seconds** without a visual change, practice pause, or interaction
- [ ] **Teach → Drill → Apply cycle** in each section (not stacked: don't teach 5 things then drill 5 things)
- [ ] **Duration realistic** for target (count ~150 words/min narration, add pause time)
- [ ] **Energy/tone cues** marked at every section transition (`[ENERGY: confident]`, `[ENERGY: calm]`, etc.)

### Engagement & Interaction
- [ ] **Practice pauses have specific prompts** — tell the student exactly what to say/write/think, then reveal the answer
- [ ] **Active recall summary** — student tries to recall key points BEFORE seeing the final summary
- [ ] **Minimum 6 practice pauses** per 10-minute script
- [ ] **No passive summary lists** — always test before reveal

### Teaching Quality
- [ ] **Malayalam bridges are structural parallels**, not dictionary translations (bad: "Haus = veedu"; good: "German articles work like Malayalam's -aal suffix — they tell you something about the noun")
- [ ] **One clear teaching goal per section** — don't overload
- [ ] **Exam connection explicit** — which exam section, what skill, how many points
- [ ] **No A2+ content** leaking into A1 scripts
- [ ] **German grammatically verified** — all example sentences correct

### Production Readiness
- [ ] **Visual cues are producible** and tagged with `[SLIDE:]`, `[DIAGRAM:]`, `[FORM:]`, `[DIALOGUE:]`
- [ ] **Exam tips limited to 3 inline `[EXAM-TIP]` markers** (don't over-flag)
- [ ] **Outro has clear next-lesson bridge** — what comes next and why
- [ ] **Homework is specific and doable** — not vague "practice more"

### Script Metadata
- [ ] **Status updated** to `REVISED — v2.0` (or higher)
- [ ] **Revision notes section** documenting what changed from v1.0
- [ ] **Script Quality Checklist** appended at bottom (filled in)

---

## Common v1.0 → v2.0 Fixes

| Issue | v1.0 Pattern | v2.0 Fix |
|-------|-------------|----------|
| Talky hooks | 60-90s of motivation speech | Cut to 35-45s, show the concrete outcome immediately |
| Passive summaries | "Here's what you learned" bullet list | Active recall: "Can you name the 3 patterns? Try now... ⏸️" |
| Dictionary Malayalam | "Haus = veedu" | Structural parallel: "Think of it like how Malayalam adds -kal for plural" |
| Stacked teaching | Teach A, B, C, then drill A, B, C | Teach A → drill A → Teach B → drill B |
| Missing energy cues | Flat narration directions | Mark energy at each section: `[ENERGY: brisk]`, `[ENERGY: warm]` |
| Vague pauses | "Practice now!" | "Say 'Ich heiße [your name]' — go! ⏸️ (3s)" |
| Weak outros | "See you next time!" | Bridge to next lesson with clear hook |
| Overlong sections | 3+ minutes without interaction | Max 90s teaching before an interaction beat |

## Quality Tiers

| Tier | Status | Criteria |
|------|--------|----------|
| **v1.0** | 🔄 DRAFT | Content complete, structure present, not yet reviewed |
| **v2.0** | 🔄 REVISED | Passes full checklist, pacing/engagement improved |
| **v3.0** | ✅ DONE | Native speaker reviewed German, test-recorded for timing, final |
