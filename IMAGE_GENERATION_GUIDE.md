# Image Generation Guide (Gemini Antigravity)

Generate these images batch by batch. After each batch is done, move the entry to the "Completed" section at the bottom.

## How to use
1. Use Gemini Antigravity to generate each image
2. Save as PNG to the path specified
3. Dimensions: **1024x576** (landscape) unless noted otherwise
4. Style: **Illustrated flat design, warm colors, NO TEXT in the image**
5. After saving, move the entry from "Pending" to "Completed"
